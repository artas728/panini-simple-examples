How to run services:

1. Install & run NATS:

    Instruction here: https://docs.nats.io/nats-server/installation

2. Install panini

   `pip install panini`

3. Run process_request

   `python3 process_request2.py`

4. Run process_response

   `python3 process_response.py`

5. Run send_request

   `python3 send_request.py`
